\# CodeAlpha EDA Project



\## Overview



This project performs Exploratory Data Analysis (EDA) on the Netflix Titles dataset using Python.



The objective is to understand content distribution, identify trends, explore ratings, analyze countries producing content, and derive insights from the dataset through visualizations.



\## Technologies Used



\* Python

\* Pandas

\* NumPy

\* Matplotlib

\* Seaborn

\* Jupyter Notebook

\## Analysis Performed



\* Dataset exploration

\* Missing value analysis

\* Movies vs TV Shows comparison

\* Country-wise content analysis

\* Content growth over years

\* Rating distribution

\* Data visualization



\## Key Insights



\* Movies dominate the Netflix catalog.

\* The United States contributes the largest amount of content.

\* Netflix content increased rapidly after 2015.

\* TV-MA is among the most common ratings.
